# Happy Birthday Celebration Page

A special interactive birthday celebration webpage created to mark a special day! This colorful, animated webpage includes:

## Features
- Animated floating cake with candles
- Floating balloons and confetti effects
- Interactive surprise button with special birthday wishes
- Sound effects for added celebration
- Responsive design that works on all devices
- Beautiful gradient backgrounds and animations

## How to Use
Simply open `index.html` in any modern web browser to enjoy the birthday celebration!

## Special Features
- Click the "Click for a surprise!" button to reveal special birthday wishes
- Move your mouse around the screen to see dynamic background effects
- Watch the animated cake gently float and the flame flicker
- Enjoy the subtle confetti falling in the background

## Technologies Used
- HTML5
- CSS3 (with animations and gradients)
- JavaScript (DOM manipulation and interactive features)
- Web Audio API for sound effects
